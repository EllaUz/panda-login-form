# 🐼 Panda Login Form

A playful, purely CSS-crafted login form featuring a cute panda holding the input fields. Built entirely for fun as a design exercise.

## 🚀 Live Demo
 
[View the Panda Login](https://ellauz.github.io/panda-login-form/)

## ✨ Features

- **100% Pure HTML & CSS:** No external images, SVGs, or JavaScript used. The entire panda is built using CSS shapes, borders, and positioning.
- **Cute, Interactive Design:** The panda's body acts as the card holding the username and password inputs.
- **Clean Form UI:** Simple, centered, and responsive input fields with a clear "LOGIN" call-to-action.
- **"For Fun" Project:** A creative way to practice CSS positioning, pseudo-elements (`::before`, `::after`), and styling form elements.

## 🛠️ Tech Stack

- HTML5 
- CSS3
